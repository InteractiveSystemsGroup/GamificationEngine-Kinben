Player Data Type

A player represents a user in the gamification application, eg. an employee of an organisation or a customer. By the creation, each player is assigned a nickname and certain roles. Each player has a list for his earned rewards, already finished Goals and finished Tasks. Points, coins and index of a level can be earned or raised by fulfilling tasks in the gamification application. Furthermore a player can have an avatar. A player can be set active or can be deactivated so that she/he cannot complete tasks. By default every created player is active until she/he is deactivated. Each player can also have a list of contacts which represent other players in the same organisation to send little presents. At a later point of time it is possible to change the password, nickname, avatar and the roles or contacts a player has.

Properties
name data type description
active boolean Checks if a player is active in the gamificated application. This is needed for example to complete a task.
belongsToRoles array of Role Gets all roles a player has.
coins number Gets the current amount of coins a player has obtained.
contactList array of Player Gets the List of all player's contacts.
finishedGoals array of FinishedGoal Gets all Goals a player has already completed.
finishedTasks array of FinishedTask Gets all Tasks a player has already finished.
id number Gets the id of a player.
levelIndex number Gets the current level index of a player.
levelLabel string Gets the label of a player's current level.
nickname string Gets the nickname of a player.
onlyAchievement array of Achievement Gets only all Achievements a player has already obtained. If the player has no Achievement, null is returned.
onlyBadges array of Badge Gets only all Badges a player has already obtained. If the player has no Badge, null is returned.
points number Gets the current amount of points a player has obtained.
reference string Gets the real name of a player.
rewards array of PermanentReward Gets all permanent rewards a player has already obtained. These are for example all badges and achievements.
